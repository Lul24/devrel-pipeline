# DevRel Pipeline — AI Content Automation

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://lul24.github.io/devrel-pipeline/)

AI-powered content pipeline for Developer Relations teams. Generate platform-optimized posts, score quality, and route through multiple LLMs.

## Features
- 🔍 GitHub repo analysis
- ✍️ Multi-platform content generation (Twitter, LinkedIn, Dev.to, Reddit, Substack)
- 📊 AI quality scoring (Clarity, Engagement, Tech Accuracy, Virality)
- 🎯 Model routing (Claude vs GPT-4o vs Gemini)
- 📦 Export to Markdown, JSON, HTML, or CSV

## Quick Start
1. Enter your Claude API key
2. Paste a GitHub URL
3. Click "Analyze Source"
4. Generate, score, and route your content
5. Download your content pack
